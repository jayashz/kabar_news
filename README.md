### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/your-username/kabar_news.git
   cd kabar_news
   ```
2. Create a file named keys.env
   create this file and add in you api key as:
   News_API_Key= **YourAPiKeyhere**
3. Run the flutter app
   ```bash
   flutter run
   ```
